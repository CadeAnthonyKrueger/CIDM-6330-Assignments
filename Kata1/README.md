# Kata-1
## Data Source

Weather data used in this project was obtained from the **NOAA National Centers for Environmental Information (NCEI)** Climate Data Online system:  
https://www.ncei.noaa.gov/access/search/index

The dataset comes from the **Global Historical Climatology Network Daily (GHCND)** product and was downloaded in **CSV format**.

### Station
- USC00112931

### Date Range
- January 1, 2026 through February 4, 2026

### Variables Included
- **PRCP** – Precipitation  
- **SNOW** – Snowfall  
- **SNWD** – Snow depth  
- **TMAX** – Daily maximum temperature  
- **TMIN** – Daily minimum temperature  
- **TOBS** – Observed temperature  

The dataset also includes station metadata (such as station name and geographic coordinates) and NOAA data flags associated with each measurement.

### Units
- Data is provided in NOAA standard units.  
- Temperature values are stored in **tenths of degrees Celsius**, following GHCND conventions.
